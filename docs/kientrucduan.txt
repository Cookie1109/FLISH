## Kiến trúc tổng thể

Dự án được chia thành 3 tầng chính: **Client** (Handlebars + Web Speech API), **Server** (Express + Route Handler + Event-driven), và **Data** (PostgreSQL + Sequelize). Docker compose sẽ orchestrate toàn bộ.

## Event-driven Pattern

Node `EventEmitter` được dùng để **tách biệt side-effects** khỏi business logic — ví dụ khi user xem một card, server emit event `card.viewed`, một listener riêng sẽ cập nhật bảng `learning_progress` mà không làm chậm response chính. Tương tự với `quiz.completed` để tính điểm và `card.created` để log audit.

## Luồng tạo Flashcard

Khi user nhập một từ mới, server sẽ song song gọi **Free Dictionary API** (lấy phonetic + definition + câu ví dụ) và **Unsplash API** (lấy hình ảnh minh họa), sau đó gọi **MyMemory API** để dịch câu ví dụ sang tiếng Việt. Toàn bộ dữ liệu được lưu vào PostgreSQL trong một transaction.
